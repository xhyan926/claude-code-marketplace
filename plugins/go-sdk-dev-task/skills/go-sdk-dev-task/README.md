# go-sdk-dev-task Plugin

华为云 OBS Go SDK 开发任务分解技能。

## 技能说明

`go-sdk-dev-task` 技能用于将 OBS SDK Go 开发任务分解为可独立完成、可追踪、具有幂等性的子任务。

## 功能特性

- 标准化任务分解
- 完整的子任务模板
- 与 /go-sdk-ut skill 集成
- 幂等性保证
- 验收报告生成

## 使用方法

在对话中使用技能会自动触发：

```
"我需要开发一个新的 OBS 功能，怎么分解这个任务？"
"帮我规划一下这个重构任务"
```

详细使用示例请参阅 `skills/go-sdk-dev-task/USAGE_EXAMPLES.md`。
